앱 시나리오 및 화면 구성
1. 로그인 페이지 - 틴더 로그인 페이지 참고(페이스북 + 카카오톡)
2. 메인화면 - 하트 페이지
	2.1 하트 페이지에서 본인 사진 표시(자기프로필 사진 변경 기능(로컬이미지) : 프로필 사진 서버 저장), 
	    상대방이 등록 되지 않았을 때 상대방 사진에 INVITE 표시
	    상대방 사진을 눌러서 초대(페이스북, 카카오톡, 연락처 목록에서 선택 가능하도록)
	2.2 메뉴를 통해서 하트 페이지 추가 가능하도록 하는 기능 - 2차 버전 기능으로 1차 버전 제외
	2.3 하트 기능
		2.3.1 상대방이 등록되지 않을 경우 하트 버튼 누를 때 상대방 초대 팝업
		2.3.2 상대방이 등록 된 후 하트 버튼 누를 때 서버에 하트 전송
		2.3.3 하트 전송 후 매칭 대기 애니메이션(60초)
		2.3.4 대기 시간 내에 매칭 성공시 매칭 성공 애니메이션 -> 초기화면
		2.3.5 대기 시간 내에 매칭 실패시 하트 크래쉬(매칭 실패 애니메이션) -> 초기화면
		2.3.6 하트 전송 후 다음 하트 보낼 수 있는 시간(기본 사용 가능한 하트 10개, 10분 후 추가 충전, 뱅크 확장 가능 기본 10개)
	2.4 매칭 성공 결과 정보를 하단(또는 상단)에 숏컷으로 표시(총 누적 : 다단계 하트)
